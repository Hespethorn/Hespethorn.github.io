C语言标准库函数复现与深度解析

本文通过手动实现6个标准C库字符串函数，深入剖析其底层原理和实现细节。所有代码保持原始实现，重点分析潜在问题并提供改进建议。

🔍 功能概览

函数名 功能描述 核心参数 返回值类型
my_strlen 计算字符串长度 const char* 指针 size_t
my_strcpy 完整字符串复制 dest, src 双指针 char*
my_strncpy 安全长度限制复制 dest, src, n 三参数 char*
my_strcat 字符串追加 dest, src 双指针 char*
my_strncat 安全长度限制追加 dest, src, n 三参数 char*
my_strcmp 字符串比较 const char* 双指针 int

🛠️ 函数实现解析

1. my_strlen - 字符串长度计算

关键技术点：

  • 通过指针自增遍历字符数组
  • 严格依赖\0终止符
  • 时间复杂度O(n)

潜在问题：

  • 未处理空指针输入（需添加NULL检查）

2. my_strcpy - 字符串复制

执行流程：

  1. 保存目标指针用于返回
  2. 逐字节复制源字符串
  3. 强制添加终止符

安全缺陷：

  • 无缓冲区大小检查，存在溢出风险

3. my_strncpy - 安全复制

改进亮点：

  • 添加长度限制参数n
  • 自动填充终止符

边界情况：

  • 当src长度<n时：剩余空间填充'\0'
  • 当src长度≥n时：不保证终止符

4. my_strcat - 字符串追加

执行过程：

  1. 定位目标字符串末尾
  2. 追加源字符串内容
  3. 自动添加终止符

安全隐患：

  • 目标缓冲区空间不足可能导致溢出

5. my_strncat - 安全追加

关键改进：

  • 限制最大追加长度
  • 确保字符串终止

注意要点：

  • 需要同时检查源字符串结束和n的限制

6. my_strcmp - 字符串比较

比较规则：

  • 逐字符ASCII值比较
  • 返回值为：
    • 负数：str1 < str2
    • 0：相等
    • 正数：str1 > str2

符号处理：

  • 使用unsigned char避免符号扩展问题
